\textbf{Problem 5: Overlapping sums}

Given: $X_1, X_2, \ldots$ are independent exponential(2) random variables.
- $X = X_1 + X_2 + \cdots + X_n$ (sum of first $n$ terms)
- $Y = X_{n-7} + X_{n-6} + \cdots + X_{2n-8}$ (sum from $X_{n-7}$ to $X_{2n-8}$)
- Assume $n \geq 8$

\textbf{Step 1: Identify the overlap}

$X$ includes indices: $1, 2, \ldots, n$

$Y$ includes indices: $n-7, n-6, \ldots, 2n-8$

The overlap occurs for indices $i$ where $\max(1, n-7) \leq i \leq \min(n, 2n-8)$.

Since $n \geq 8$:
- $n-7 \geq 1$, so $\max(1, n-7) = n-7$
- $2n-8 \geq n$ (since $n \geq 8$), so $\min(n, 2n-8) = n$

Therefore, the overlapping indices are: $n-7, n-6, n-5, n-4, n-3, n-2, n-1, n$.

Number of overlapping terms = $8$.

\textbf{Step 2: Calculate covariance}

Using linearity of covariance:
$$\text{Cov}(X, Y) = \text{Cov}\left(\sum_{i=1}^n X_i, \sum_{j=n-7}^{2n-8} X_j\right) = \sum_{i=1}^n \sum_{j=n-7}^{2n-8} \text{Cov}(X_i, X_j)$$

Since the $X_i$ are independent:
$$\text{Cov}(X_i, X_j) = \begin{cases}
0 & \text{if } i \neq j \\
\text{Var}(X_i) = \frac{1}{2^2} = \frac{1}{4} & \text{if } i = j
\end{cases}$$

Therefore:
$$\text{Cov}(X, Y) = \sum_{i=n-7}^n \text{Var}(X_i) = \sum_{i=n-7}^n \frac{1}{4} = 8 \cdot \frac{1}{4} = 2$$

\textbf{Step 3: Calculate correlation}

First, find the variances:
$$\text{Var}(X) = \sum_{i=1}^n \text{Var}(X_i) = n \cdot \frac{1}{4} = \frac{n}{4}$$

The number of terms in $Y$ is $(2n-8) - (n-7) + 1 = n$, so:
$$\text{Var}(Y) = n \cdot \frac{1}{4} = \frac{n}{4}$$

Therefore:
$$\text{Cor}(X, Y) = \frac{\text{Cov}(X, Y)}{\sqrt{\text{Var}(X) \cdot \text{Var}(Y)}} = \frac{2}{\sqrt{\frac{n}{4} \cdot \frac{n}{4}}} = \frac{2}{\sqrt{\frac{n^2}{16}}} = \frac{2}{\frac{n}{4}} = \frac{8}{n}$$

\textbf{Final Answer:}
$$\boxed{\text{Cov}(X, Y) = 2, \quad \text{Cor}(X, Y) = \frac{8}{n}}$$
