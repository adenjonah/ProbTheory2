\textbf{Problem 1(a): Find $c$ and the joint cdf $F(x,y)$.}

To find $c$, we use the condition that the joint pdf must integrate to 1:
$$\int_0^1 \int_0^1 f(x,y) \, dx \, dy = 1$$

$$\int_0^1 \int_0^1 c(x^2 + xy) \, dx \, dy = c \int_0^1 \int_0^1 (x^2 + xy) \, dx \, dy$$

$$= c \int_0^1 \left[ \frac{x^3}{3} + \frac{x^2y}{2} \right]_0^1 dy = c \int_0^1 \left( \frac{1}{3} + \frac{y}{2} \right) dy$$

$$= c \left[ \frac{y}{3} + \frac{y^2}{4} \right]_0^1 = c \left( \frac{1}{3} + \frac{1}{4} \right) = c \cdot \frac{7}{12}$$

Setting this equal to 1: $c \cdot \frac{7}{12} = 1$, so $c = \frac{12}{7}$.

Therefore: $f(x,y) = \frac{12}{7}(x^2 + xy) = \frac{12x^2}{7} + \frac{12xy}{7}$ for $(x,y) \in [0,1] \times [0,1]$.

For the joint cdf $F(x,y) = P(X \leq x, Y \leq y)$:
$$F(x,y) = \int_0^y \int_0^x f(u,v) \, du \, dv = \int_0^y \int_0^x \frac{12}{7}(u^2 + uv) \, du \, dv$$

$$= \frac{12}{7} \int_0^y \left[ \frac{u^3}{3} + \frac{u^2v}{2} \right]_0^x dv = \frac{12}{7} \int_0^y \left( \frac{x^3}{3} + \frac{x^2v}{2} \right) dv$$

$$= \frac{12}{7} \left[ \frac{x^3v}{3} + \frac{x^2v^2}{4} \right]_0^y = \frac{12}{7} \left( \frac{x^3y}{3} + \frac{x^2y^2}{4} \right)$$

$$F(x,y) = \frac{4x^3y}{7} + \frac{3x^2y^2}{7} = \frac{x^2y(4x + 3y)}{7}$$
