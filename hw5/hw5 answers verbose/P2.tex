\noindent
\textbf{Problem 2: Lognormal Distribution}

\vspace{0.5em}
\noindent
If $X$ has the lognormal distribution with parameters $\mu = 3$ and $\sigma^2 = 1.44$, then $\log(X) \sim N(3, 1.44)$.

\vspace{0.5em}
\noindent
We seek:
$$P(X \leq 6.05) = P(\log(X) \leq \log(6.05))$$

Since $\log(X) \sim N(3, 1.44)$ with $\sigma = \sqrt{1.44} = \frac{6}{5} = 1.2$:
$$P(\log(X) \leq \log(6.05)) = \Phi\left(\frac{\log(6.05) - 3}{1.2}\right)$$

Computing the standardized value:
$$z = \frac{\log(121/20) - 3}{6/5} = -\frac{5}{2} + \frac{5\log(121/20)}{6} \approx -0.9999514$$

Therefore:
$$P(X \leq 6.05) = \Phi(-1.0000) \approx \boxed{0.1587}$$

\vspace{1em}
\noindent
\textbf{Verification:}
\begin{itemize}
    \item Exact parameters: $\mu = 3$, $\sigma^2 = \frac{36}{25}$, $\sigma = \frac{6}{5}$
    \item Standardized value: $z = -\frac{5}{2} + \frac{5\log(121/20)}{6} \approx -0.9999514$
    \item Theoretical probability: $P(X \leq 6.05) = 0.158667$
    \item Verification using \texttt{scipy.stats.lognorm}: 0.158667 (exact match)
    \item Monte Carlo verification (n=10,000,000): 0.158828
    \item Difference: $1.61 \times 10^{-4} < 10^{-3}$ (pass)
    \item Random seed: 123456
\end{itemize}
